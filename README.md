**Biomathematics**
This is a section of my work devoted to the use of mathematical tools to model, analyse, and simulate biological phenomena. The associated codes, available here, illustrate how differential equations, compartmental models, partial differential equations, and numerical methods can be used to study population dynamics, disease spread, cellular dynamics, and other biological processes. This section highlights the role of mathematics in understanding living systems by making it possible to formulate models, estimate parameters, perform simulations, and interpret the results obtained. The main objective is to show how biomathematics serves as a powerful tool for connecting mathematical theory with concrete applications in biology, epidemiology, and public health.
